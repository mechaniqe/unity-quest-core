# Quest Events Sample

Demonstrates subscribing to quest system events for UI updates or game logic.

## What It Shows

- `OnQuestCompleted` - Triggered when a quest finishes successfully
- `OnQuestFailed` - Triggered when a quest fails
- `OnObjectiveStatusChanged` - Triggered when any objective status changes
